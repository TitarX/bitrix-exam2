1

Создаём инфоблок для жалоб, два свойства, устанавливаем права на чтение для всех.
Меняем отображение списка элементов инфоблока жалов в админке.

--------

2

В шаблоне комплексного компонента news, и в шаблоне компонента news.detail, создаём параметр включения Ajax "Собирать жалобы в режиме AJAX", пробрасываем параметр в detail.php.

В шаблоне компонента news.detail, файл template.php, возле заголовка h3, создаём ссылку "Пожаловаться!". Ссылку на текущую страницу, с параметрами признака жалобы и идентификатора элемента. По условию, при оключенном параметре "Собирать жалобы в режиме AJAX", создаётся обычная ссылка с параметрами на текущую страницу; при включенном параметре "Собирать жалобы в режиме AJAX", создаётся ссылка и обработчик клика с кодом отправки запроса Ajax на текущую страницу.
Рядом со ссылкой создаём элемент, в котором будет выводится результат запроса добавления жалобы.
В начало файла template.php добавляем 'CJSCore::Init(array("ajax"));', для подключения ядра JS-библиотеки Bitrix и библиотеки Ajax.

--------

3

В шаблоне компонента news.detail создаём файл component_epilog.php, в котором будет код обработки обычного запроса и Ajax.
В начало файла template.php добавляем 'CJSCore::Init();', для подключения ядра JS-библиотеки Bitrix.
